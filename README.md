# Stock Opname Web

Aplikasi Stock Opname sederhana:
- Scan barcode (scanner USB / kamera)
- Ambil nama produk dari Google Sheet
- Input qty
- Simpan hasil SO

## Cara Pakai
1. Isi nama petugas
2. Pilih metode scan
3. Scan produk
4. Input qty
5. Simpan

## Tech
- GitHub Pages
- Google Apps Script
- Google Sheets
